# Appointment Booking Android Application

Android application for online appointment booking and management between professionals and customers.

## Overview

This project was developed as a diploma thesis at the Department of Computer Science and Engineering, University of Ioannina.

The application allows professionals to create and manage their profile, services, weekly schedule and appointments. Customers can search for professionals, browse profiles, add services to a cart and book appointments through a mobile Android interface.

## Features

- Professional and customer registration/login
- Professional profile and business information management
- Service categories and service management
- Weekly schedule configuration
- Appointment booking and management
- External appointment entry for customers outside the platform
- Customer search by profession, specialty and location
- Service cart before appointment booking
- Appointment status updates
- Google Maps and WhatsApp integration

## Tech Stack

- Java
- Android Studio
- XML layouts
- PHP
- MySQL
- PHPMyAdmin
- Online server-hosted database

## Architecture

The application follows a client-server architecture. The Android application acts as the client, while PHP scripts handle communication with a MySQL database hosted on an online server.
